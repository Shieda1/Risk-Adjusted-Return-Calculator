What is the risk-adjusted return?
Risk-adjusted return is a method used by investors to measure the profit their investments has made relative to the risk these invesments has represented, so if there have been tow investments made in the same time period, the one that has the lowest risk will have a better risk-adusted return.
How to calculate the risk-adjusted return?
To calculate the risk-adjusted return you should use the following equation:
Risk-adjusted return = (portfolio return - risk free return) / standard deviation of portfolio return
Why use the risk-adjusted return?
Investors use the risk-adjusted return to measure the risk in relation to the profit to make the best investments they can do in that specific time period.
